# Тестовое задание VueJS

Рабочая страница с изображениями для выбора вещей.

## Описание

Приложение позволяет:
- Выбирать от 1 до 6 вещей из блока "Вещи у пользователя" (снизу слева)
- Выбирать только 1 вещь из блока "Вещи на выбор" (снизу справа)
- Отображать выбранные элементы в соответствующих верхних блоках

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build

# Просмотр production сборки
npm run preview
```

## Функциональность

- **Блок снизу слева**: Вещи у пользователя (можно выбрать от 1 до 6)
- **Блок снизу справа**: Вещи на выбор (можно выбрать только 1)
- **Блок сверху слева**: Отображает выбранные вещи пользователя в порядке выбора
- **Блок сверху справа**: Отображает выбранную вещь на выбор
